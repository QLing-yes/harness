## [PROMPT-001] 任务拆解

**使用时机：** 有一个模糊的大任务，需要拆成可执行的子任务
**验证日期：** {{YYYY-MM-DD}}

```
我有一个任务：{{任务描述}}

请帮我拆成 3-5 个具体的子任务，每个子任务要：
1. 有明确的完成标准（"写完"不算，"通过 X 测试"才算）
2. 可以独立执行
3. 预计在 2 小时内能完成

以 Markdown 清单格式输出。
```

## [PROMPT-002] 复盘生成

**使用时机：** 任务完成或出错后，快速生成 feedback 记录
**验证日期：** {{YYYY-MM-DD}}

```
我刚完成了：{{任务描述}}

过程中出现了这个问题：{{问题描述}}

请帮我生成一条 feedback 记录，格式：
- 发生了什么（客观描述）
- 根本原因
- 已采取的修复
- 预防措施（要具体，比如"在 [编码规范](./rules.md) 添加 X 规则"）
```
